(static) by(list, n, n) → {void}

Source:
  • utilities.js, line 36

Iterates and calls the callback parameter for each element or property of a list at the interval specified by the n parameter. It does not call callback on values greater than the list’s number

Example

const double = (x) => result.push(x * 2);

(static) pick(obj, keys) → {Object}

Source:
  • utilities.js, line 271

Returns a new object by picking all key/value pairs from the argument obj. The keys that are picked will be determined by the array parameter keys.

Example
var data = {
    type: "transformer",
    index: 19,
    siblings: 19,
    access: "full"
};

// returns {type: "transformer", index: 19};
sjs.U.pick(data, ["type", "index"]);

// returns {siblings: 19, index: 19};
sjs.U.pick(data, ["siblings", "index"]);

// returns {access: "full"};
sjs.U.pick(data, ["access", "animals"]);

(static) pluralize(n, word, pluralWord) → {String}

Source:
  • utilities.js, line 158

Returns the plural of a word depending on the value of the n parameter. If n is 1, return the non-plural word (parameter word); otherwise, add an “s” to the plural word. If the pluralWord parameter is provided, instead of adding an “s,” return the pluralWord.

Examples
// returns "lion"
sjs.U.pluralize(1, "lion");
// returns "lions"
sjs.U.pluralize(2, "lion");
// returns "lions"
sjs.U.pluralize(5, "lion");
// returns "lions"
sjs.U.pluralize(0, "lion");
// returns "lioness"
sjs.U.pluralize(1, "lioness");
// returns "lionesss"
sjs.U.pluralize(2, "lioness");
// "lionesses"
sjs.U.pluralize(2, "lioness", "lionesses);
Parameters:
Name Type Description
n int

The number of "s"
word String

A non-plural word
pluralWord String

An optional plural word
Returns:

A pluralized string

Type
String
